Problem 2: The p.adjust Function (4 pts total)

Consider the following set of five p-values:

sim_pvalues2 <- c(0.02, 0.03, 0.06, 0.15, 0.25)

Part a (Code and Explanation: 1 pt)

Apply the p.adjust function with method = "bonferroni" to this set of p-values. Explain precisely what this method does. Pay special attention to what p.adjust does with the value 0.25.

Solution

p.adjust(sim_pvalues2, method = "bonferroni")
[1] 0.10 0.15 0.30 0.75 1.00

What Bonferroni does is returns p values after being multiplied by the number of comparisons. In this case, 5 p-values, so each p-value is multiplied by 5, with .25 capping at 1.00 since p-values are probabilities.

Part b (Code and Explanation: 1 pt)

Apply the p.adjust function with method = "holm" to this set of p-values. Explain precisely what this method does. Pay special attention to the last two p-values.

Solution

p.adjust(sim_pvalues2, method = "holm")
[1] 0.10 0.12 0.18 0.30 0.30

What Holm does is before a p-value is returned is similar to Bonferroni, except instead of multiplying by just the number of comparisons, it is multiplied by that said number + 1 - i, with i being the index of the order of the affected p-value. Though, the final value should be .25 following this, the reason it is .30 is because in this method, p-values must stay in order from small to big, hence, to retain validity, .25 must be adjusted to at least the same value as the p-value before, which is .30. This method is less conservative than the last and retains a greater power which means fewer Type II Errors.

Part c (Code and Explanation: 1 pt)

Apply the p.adjust function with method = "BH" to this set of p-values. Explain precisely what this method does. Pay special attention to the first two p-values.

Solution

p.adjust(sim_pvalues2, method = "BH")
[1] 0.0750 0.0750 0.1000 0.1875 0.2500

What BH does is very similar to Holm, except it multiplies the p-value by the quantity, and divides by the placement of the p-value. Another difference is that .02 was sized down to be equivalent to .075. So when the adjusted p-value is bigger than the value to it’s right, it is sized down to an equivalent. This method is less conservative than the last and retains a greater power which means fewer Type II Errors.

Their group assignment, and alcohol intake over the year (in centiliters of pure alcohol), is found in the alcohol.csv dataset on Canvas.

Using a permutation t-test, we wish to determine whether there is convincing evidence of a difference in population mean alcohol consumption between patients using the SST program and patients using a traditional treatment program.

Part a (Explanation: 1 pt)

Specify the null and (two-sided NHST) alternative hypothesis using both words and math.

Solution

The null hypothesis would be that there is no difference in effectiveness between the two programs. The alternative is that a difference between them exists.

H0: mu1 = mu2 & Ha: mu1 /= mu2
